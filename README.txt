How to Compile ?

To compile and run, all you need to do is run the Solution.java file. Running the Solution.java file in the src will play 5 rounds of the game and print the results in the output.txt file.


From this simulation of the game of Craps, it is evident that predicting the best strategy for a game of chance is very infeasible because in such games it is not possible to get a certain pattern.
